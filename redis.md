#Redis

**Présentation :** base de données stockées dans la mémoire vive

**Help :**
- *redis-cli -h **IP*** : connexion
- *info* : donne les infos
- *select **INDEX*** : pour sélectionner la base. ex : *db2 => select 2*
- *keys \** : liste toutes les clés de la table
